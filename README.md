# 🌦 Weather App – Real-Time Weather Information Using OpenWeather API

A Python-based Weather Application that fetches real-time weather information using the OpenWeather API. This project demonstrates API integration, secure environment variable management, error handling, and Python project structuring.

---

## 📌 Project Status

✅ Version 1.0 Completed

🚀 Future enhancements and additional features are planned.

---

## 🎯 Project Overview

This application allows users to search for weather information of any city and receive real-time weather details such as temperature, humidity, and weather conditions.

The project was built to gain practical experience with:

* API Integration
* Python Development
* Environment Variable Management
* Error Handling
* Git & GitHub Workflow
* Project Documentation

---

## ✨ Features

* Search weather by city name
* Real-time weather information
* Temperature display in Celsius
* Humidity percentage display
* Weather condition display
* Error handling for invalid city names
* Secure API key management using `.env`
* Clean and beginner-friendly code structure

## ⚙️ Installation

### 1. Clone the Repository

```bash
git clone https://github.com/nikhilghotankar-8881/Weather-App.git
```

### 2. Navigate to Project Directory

```bash
cd Weather-App
```

### 3. Install Dependencies

```bash
pip install -r requirements.txt
```

### 4. Create a `.env` File

```env
API_KEY=your_openweather_api_key
```

### 5. Run the Application

```bash
python main.py
```
